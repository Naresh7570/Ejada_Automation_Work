# UI API Framework

Maven-based automation framework for UI and API tests.

## Project Structure

```text
src
├── main
│   └── java
│       ├── pages
│       ├── utils
│       └── drivers
└── test
    ├── java
    │   ├── stepdefinitions
    │   ├── runners
    │   └── tests
    └── resources
        ├── features
        └── config.properties
```

## Run Tests

```bash
mvn test
```
